Ejercicios

Preguntas

  1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?
    Java es un lenguaje multiproceso compilado, mientras JavaScript es un lenguaje interpretado (Se interpreta mientras se ejecuta), también suele usarse para crear páginas web interactivas. Su similitud es principalmente de sintaxis.
  2. ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)
    getDate(), getMonth(), getFullYear(), getHours(), getMinutes()
  3. ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)
    push(), pop(), concat(), map(), slice()
  4. ¿Cómo se declara una variable con alcance local dentro de una función?
    Con let, pero si se desea que no cambie debe usarse const.
  5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?
    Hay mayor riesgo de sobreescritura, también es accesible para todo el código e impacta el rendimiento.
